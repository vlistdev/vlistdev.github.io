User Data Deletion Method

During App Usage

  • Tap the target checklist from the "Checklist Publishing" screen to display the checklist details screen.
  • Tap the three-dot menu in the upper-right corner of the checklist details screen and select "Delete."
  • All user data associated with the account will be deleted upon account deletion.

Outside of App Usage

  • If there is no login activity for 180 days, the account will be automatically deleted along with all associated user data.
